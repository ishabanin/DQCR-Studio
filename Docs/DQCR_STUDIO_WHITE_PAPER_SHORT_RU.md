# DQCR Studio White Paper

## Что это

**DQCR Studio** - это единая рабочая среда для разработки, сопровождения, проверки и сборки DQCR-проектов. Продукт объединяет файловую структуру проекта, SQL/YAML-редактирование, визуальную работу с моделями, продвинутый autocomplete, поддержку внешнего каталога данных, анализ зависимостей, validate и build в одном интерфейсе.

## Для кого

- Data Engineer / Analytics Engineer
- DQCR-архитектор / техлид
- Аналитик / power user
- Администратор платформы

## Какую задачу решает

DQCR Studio убирает разрыв между файлами проекта, framework CLI и пользовательским интерфейсом. Вместо набора папок, ручных команд и сложной навигации команда получает управляемую среду, где можно быстрее вносить изменения, проверять их и выпускать результат.

## Основной функционал

- Project Hub для управления проектами
- Workbench с файловым деревом проекта
- SQL Editor и YAML/visual model editing
- продвинутый autocomplete по объектам проекта и каталога данных
- работа с контекстами и параметрами
- Lineage и анализ зависимостей
- Validate с ошибками, предупреждениями и quick fixes
- Build и просмотр артефактов
- поддержка внешнего каталога данных и импорт сущностей в модели
- администрирование каталогов, шаблонов и правил

## 4 Killer Features

### 1. Filesystem-first

Источником истины остается реальная файловая структура проекта. Это упрощает Git-процессы, прозрачность и сопровождение.

### 2. Один цикл работы

Редактирование, анализ, validate и build собраны в одном продукте. Команда проходит путь от изменения до результата без переключения между разными инструментами.

### 3. Прозрачность сложного проекта

DQCR Studio показывает зависимости, параметры, контексты, объекты каталога данных и артефакты сборки в понятном виде, снижая зависимость от экспертов, которые держат знания "в голове".

### 4. Внешний каталог данных + умный autocomplete

DQCR Studio поддерживает внешний каталог данных, поиск сущностей и импорт атрибутов в модели. На этой основе продукт дает продвинутый autocomplete по объектам проекта и каталога, ускоряя разработку и снижая число ошибок.

## Итог

DQCR Studio помогает командам разрабатывать DQCR-проекты быстрее, безопаснее и прозрачнее, сокращая время на онбординг, диагностику и выпуск изменений.
